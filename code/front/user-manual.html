Getting Started

Welcome to Quickest Path! This guide explains how to use our site and highlights the powerful algorithm behind it. Whether you’re a first-time visitor or a returning user, these steps will help you make the most of our service.

Step-by-Step Instructions

  1. Open the Home Page: Visit our main page and locate the Start and End fields, which accept integers from 1 to 23,947,347.
  2. Enter Valid Integers: Ensure your inputs are within the allowed range.
  3. Send Request: Click the Send Request button to process your input.
  4. View Results: A JSON response displays the calculated quickest path. Use it to make informed decisions or simply explore the route connections.

Frequently Asked Questions

1. What if I enter a number outside the valid range?
You’ll see an error message advising you to stick to numbers between 1 and 23,947,347.
2. How does the algorithm determine the path?
We analyze possible routes using heuristics and optimized data structures to quickly find an optimal route.
3. Can I submit multiple requests rapidly?
Yes, though the button is briefly disabled during processing to avoid accidental double submissions.
